HTML Typography, all you need to know

1. <h1> a <h6> (Tags de Título)

Estrutura Semântica e Otimização de SEO

Use tags de título (<h1> a <h6>)para criar uma estrutura hierárquica para o seu conteúdo, com <h1> sendo o mais importante e <h6> sendo o menos. Garanta que cada página tenha apenas uma tag <h1>, representando o título principal da página. Use tags de título para fornecer contexto e melhorar a acessibilidade para leitores de tela e mecanismos de busca.

Exemplo: html Copiar código. 

       
            <h1>Main Heading</h1> 
            <h2>Subheading </h2>
            <p>Content...</p>
            <h3>Subheading 2</h3>
            <p>Content...</p>

2. <p> (Paragraph)

Use a tag <p> para representar parágrafos de texto dentro do seu conteúdo. Aplique estilos CSS aos elementos <p> para controlar o alinhamento do texto, altura da linha e espaçamento para melhorar a legibilidade. Use a tag <p> com estilos inline ou classes para criar formatação de parágrafo visualmente atraente e consistente.

Exemplo:

<p style="text-align: justify; line-height: 1.5;">This is a paragraph with custom styling.</p>
<p style="highlight">This is a paragraph with custom styling.</p>

3. <hr> (Regra Horizontal)

Separação Visual e Estilização

Use a tag <hr> para inserir uma regra horizontal, criando separação visual entre seções de conteúdo. Aplique estilos CSS para personalizar a aparência da regra horizontal, como cor, largura e estilo. Combine as tags <hr> com outros elementos HTML e CSS para criar designs de layout complexos.

Exemplo:

<hr style="border-color: #ccc; border-width: 2px;">

4. <blockquote> (Citação em Bloco)

Citação de Conteúdo Externo

Use a tag <blockquote> para representar citações estendidas de fontes externas. Inclua o atributo cite para fornecer a fonte da citação. Aplique estilos CSS aos elementos <blockquote> para diferenciá-los do texto comum e fornecer ênfase visual.

Exemplo:

<blockquote cite="https://www.example.com/">
    Esta é uma citação em bloco de uma fonte externa.
</blockquote>

5. <pre> (Texto Pré-formatado)

Preservação de Espaço em Branco e Formatação

Use a tag <pre> para preservar espaço em branco e formatação dentro do texto, como trechos de código ou arte ASCII. Combine a tag <pre> com a tag <code> para representar blocos de código com destaque de sintaxe. Aplique estilos CSS para personalizar a aparência do texto pré-formatado, como família de fontes e cor de fundo.

Exemplo: 

        
            <pre>
               function greet()  {
                    console.log("Olá, mundo!")
               }
            </pre>

6. <abbr> (Abreviação)

Caso de Uso : Fornecimento de Definições

Use a tag <abbr> para marcar abreviações ou siglas dentro do seu conteúdo. Inclua o atributo title para fornecer a expansão completa ou definição da abreviação. Aplique estilos CSS aos elementos <abbr> para distingui-los visualmente do texto comum.

Exemplo:

<abbr title="Hypertext Markup Language">HTML</abbr> is used for creating web pages.

7. <address> (Informações de Contato)

Caso de Uso: Representação Semântica de Informações de Contato

Use a tag <address> para representar informações de contato, como endereços postais ou endereços de e-mail. Inclua texto ou outros elementos HTML dentro da tag <address> para fornecer detalhes de contato adicionais. Aplique estilos CSS aos elementos <address> para controlar sua aparência e alinhamento dentro da página.

Exemplo: 

        
            <address>
            João Silva
            Rua Principal, 123
            Cidade, Estado CEP
            Email: joao@exemplo.com
            </address>

8. <cite> (Citação)

Caso de Uso: Referenciando Obras

Use a tag <cite> para marcar o título de uma obra criativa, como um livro, filme ou artigo. Inclua o título da obra dentro da tag <cite> para fornecer significado semântico e melhorar a acessibilidade. Aplique estilos CSS aos elementos <cite> para distingui-los visualmente do texto comum, como itálico ou adição de aspas.

Exemplo:

<p><cite>O Grande Gatsby</cite> de F. Scott Fitzgerald é um romance clássico.<p>

9. <code> (Código)

Caso de Uso : Representando Trechos de Código

Use a tag <code> para representar trechos de código inline dentro do seu conteúdo. Combine a tag <code> com a tag <pre> para exibir blocos maiores de código. Aplique estilos CSS aos elementos <code> para personalizar a aparência dos trechosde código, como família de fontes e cor de fundo.

Exemplo:

<p>Use a função <code>print()</code>para exibir saída em Python.</p>

10. <strong> (Ênfase Forte e Ênfase)

Use a tag <strong> para representar texto com forte importância, como palavras-chave ou conteúdo enfatizado. Use a tag <em> para representar texto com ênfase, como palavras ou frases enfatizadas. Aplique estilos CSS aos elementos <strong> <em> para controlar sua aparência, como peso da fonte e estilo.

Exemplo:

<p> <strong>Importante:</strong> Esta informação não deve ser ignorada. </p>
<p> <em>Cuidado:<em> Prossiga com cuidado ao executar esta operação. </p>
